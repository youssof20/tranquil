# Tranquil – All-in-One Distraction Remover

Tranquil is a lightweight browser extension that helps you reclaim your focus by hiding feeds, recommendations, and other distractions on popular social platforms.

## Supported Sites

- **YouTube**: Hide home feed, sidebar, comments, and shorts.
- **Reddit**: Hide main feed, sidebar, and trending.
- **Instagram**: Hide feed, reels, and explore.
- **X (Twitter)**: Hide 'For You' feed, trends, and suggestions.
- **LinkedIn**: Hide feed, news, and recommendations.
- **Facebook**: Hide feed, stories, and suggestions.
- **Amazon**: Hide sponsored products and recommendations.

## Features

- **Live Toggles**: Enable/disable features instantly without refreshing the page.
- **Surgical Hiding**: Hides specific distracting elements while keeping the site's utility (like search) functional.
- **Pause Mode**: Temporarily pause the extension when you need a break.
- **Grayscale Mode**: Drain the colors from distracting sites to make them less addictive.
- **Privacy First**: Runs entirely on your device. No data collection, no tracking, and no external servers.

## Installation

1. Download or clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the extension folder.
